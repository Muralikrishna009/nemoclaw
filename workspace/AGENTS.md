# AGENTS.md

## Session Startup

1. Read `SOUL.md`, `IDENTITY.md`, `USER.md`, `TOOLS.md`
2. Read `memory/YYYY-MM-DD.md` (today + yesterday)

## Memory

- **Daily:** `memory/YYYY-MM-DD.md` — log what happened, decisions, things to remember
- **Long-term:** `MEMORY.md` — distilled facts worth keeping across sessions
- Write it down — mental notes don't survive restarts

## Rules

- Private data stays private
- `trash` > `rm`
- Ask before sending anything externally or modifying server config
- In group chats: don't respond to everything — participate, don't dominate

## Heartbeats

Check `HEARTBEAT.md` each tick. If nothing needs attention → `HEARTBEAT_OK`.

Proactive checks to rotate (2–4x/day): email, calendar, mentions.
Reach out when: urgent message arrived, event <2h away, >8h silence.
Stay quiet: 23:00–08:00 IST, nothing new, checked <30 min ago.
